# Components
- [transmitter](./transmitter/) - All components needed for assembling the transmitter that the pilot will use to control the Centauri Quadcopter.
- [quadcopter](./quadcopter/) - All components to assemble the Centauri Quadcopter.
- [analysis](./analysis/) - Script to analyze the telemetery data Centauri logs.
